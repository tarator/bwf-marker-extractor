BWF Marker Extractor

How it works:
  1. Upload one or more WAV files containing BWF marker data
  2. The server extracts markers using bwfmetaedit
  3. Markers are converted to Audacity-compatible label files
  4. Download the generated label files for import into Audacity (File ‚Üí Import ‚Üí Labels)

Note: This application was originally designed to extract markers generated by Zoom H5 Handy Recorder, but it should work with any BWF-compatible markers.
